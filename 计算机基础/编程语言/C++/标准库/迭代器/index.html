迭代器

一、获取迭代器​

1. 有迭代器类型的类类型​

通常有以下几个成员函数可以得到对应迭代器类型的迭代器：

  • obj：一个有迭代器类型的类类型对象
操作效果
obj.begin()返回指向obj首元素的迭代器
obj.end()返回指向obj尾后位置的迭代器
obj.cbegin()返回指向obj首元素的常量迭代器
obj.cend()返回指向obj尾后位置的常量迭代器

2. 数组类型​

可以通过iterator标准库中的函数得到类似于首位迭代器和尾后迭代器的指针：

#include <iterator>
using std::begin;
using std::end;

int arr[10];
int *beg = begin(arr);
int *end = end(arr);
无元素时两迭代器位置

首元素迭代器指向第一个元素的位置，而尾后迭代器指向尾元素后一个位置。

当容器中没有元素时，首元素迭代器和尾后迭代器将指向同一个位置，此处应该是第一个元素所在的位置但暂时没有元素。

begin() 和 end() 返回值的类型

当obj没有const进行修饰时，将返回iterator类型；而如果obj是一个经过const修饰的vector<T>对象时，将返回const_iterator类型，等价于使用cbegin()和cend()。

二、迭代器操作​

1. 通用操作​

操作效果
*iter返回迭代器iter所指元素的引用
iter->mem解引用iter并获取该元素的名为mem的成员，等价于(*iter).mem
++iter令iter指示容器的下一个元素
--iter令iter指示容器的上一个元素
iter1 == iter2
iter1 != iter2		相等性判断。当两个迭代器指向同一个位置时则相等；否则不相等

2. 迭代器运算​

如果是定义了difference_type的类型，通常可以进行以下操作：

操作效果
iter1 - iter2两迭代器相减得到它们的距离。注意有前后关系，故结果可能为负数。
运算结果是一个对应类中的difference_type类型，通常是一个整数类型
iter + n其中n应该是一个对应类中的difference_type类型，或可以转换为该类型。
假设容器从左往右分布，返回迭代器向右移动n个位置后的迭代器
iter - n其中n应该是一个对应类中的difference_type类型，或可以转换为该类型。
假设容器从左往右分布，返回迭代器向左移动n个位置后的迭代器
iter += n即iter = iter + n。iter将被改变
iter -= n即iter = iter - n。iter将被改变
iter1 < iter2
iter1 <= iter2
iter1 > iter2
iter1 >= iter2		假设容器从左往右分布，指向位置越往左越小。
