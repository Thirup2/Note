函数指针

函数指针指向的是函数而非对象，当我们把函数名作为一个值使用时，该函数自动地转换成指针：

bool lengthCompare(const string &, const string &);

auto pf = lengthCompare;        // pf 将被定义成指向 lengthCompare 这个函数类型的指针

一、函数的类型​

一个完整的函数定义可以和变量一样看成三部分：

  • 初始值：即函数体
  • 标识符：即函数名
  • 类型：除了函数体和函数名以外的函数的其他组成部分，即返回类型+参数列表

所以函数的类型包括了返回类型和参数列表这两部分，如果要声明一个指向函数类型的指针，这两部分都是必需的：

bool (*pf)(const string &, const string &);

其中pf首先是一个指针，然后*pf后紧跟参数列表，所以这是一个指向函数的指针，且返回值为bool。

二、获取函数指针​

  • 当我们把函数名作为一个值使用时，该函数自动地转换成指针
  • 在作为函数指针的初始值时，既可以直接使用函数名，也可以添加取地址符&，并没有区别

例如：

pf = lengthCompare;
pf = &lengthCompare;

和数组不同，如果直接使用数组名，那么通常将转换为指向数组首元素的指针；而如果对数组名添加取地址符&，则是得到指向数组整体的指针。

三、使用函数指针​

使用函数指针时其实和获取时类似，是否对函数指针进行解引用操作实际上影响不大，当使用函数指针调用其指向的函数时，既可以不解引用进行，也可以先解引用再进行：

bool b1 = pf("hello", "goodbye");
bool b2 = (*pf)("hello", "goodbye");
