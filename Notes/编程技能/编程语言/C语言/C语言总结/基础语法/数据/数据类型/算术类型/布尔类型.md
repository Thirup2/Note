# 一. 类型

- 类型关键词：`_Bool`
- 大小：标准保证至少可以保有两个值之一：1 和 0

另外，可以通过包含`stdbool.h`使用宏来方便使用该类型，该文件包括对`_Bool`、`1`和`0`的便利宏定义，分别是：

- `bool`：`_Bool`
- `true`：`1`
- `false`：`0`

# 二. 使用

- 声明

  ```c
  _Bool IsItTrue;
  _Bool IsItFalse;
  ```

- 初始化

  ```c
  _Bool IsItTrue = 1;
  _Bool IsItFalse = 0;
  ```

- 赋值

  ```c
  IsItTrue = 0;
  IsItFalse = 1;
  ```

- 拷贝

  ```c
  IsItTrue = IsItFalse;
  ```
