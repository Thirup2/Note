## 简介

本文档是关于数据结构与算法基础的总结，也是关于基本数据结构与算法的知识手册。基本包括大学本科阶段需要的所有内容。

本文档在数据结构部分按照数据结构的逻辑定义进行细分，你所看到的每一篇文章都是一种逻辑结构而非物理结构。因此不会单独讨论类似于顺序表、链栈、链队列的内容，它们存在于与其对应的逻辑结构的文章中。

另外，比较特别的在线性表部分原本分为一般线性表和特殊线性表（栈、队列、数组、串等），并且整个线性表的大标题应该和树、图是同一级别，但在本文档中，为了方便构建，将一般线性表和每一种特殊线性表都与树、图结构放在同一级别，并且将一般线性表特别地称为了线性表。



## 编程语言

虽然一个逻辑定义的数据结构和算法与编程语言无关，但在实现部分却需要选择具体的实现方法，则需要有一个具体的编程语言。本文档主要采用 C 语言来描述数据结构和算法，特别的地方是在列出某种数据结构的操作时会使用 C++ 中的`&`表示对对象的引用，在 C 语言中你可以将其看作该对象的指针。之所以这样做，主要是为了表示方便。
