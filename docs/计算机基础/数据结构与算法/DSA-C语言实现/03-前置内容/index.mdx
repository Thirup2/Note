---
title: 前置内容
---

为了体现数据结构与算法的抽象概念，本文档提供了一些前置库或者头文件，用来更好地让后续提供的数据结构实现与实际的编程语言尽可能地相分离，当然最基础的语法还是使用 C 语言的语法。

这些前置库包括一个数据元素类型（`elem_type`），后续的数据结构都将使用这个数据元素类型而不必用`int`、`char`等类型来代替数据元素类型，这将有利于数据结构抽象性的表现。这个数据元素类型并没有特定的实现，只提供一些接口，你可以基于这些接口自己实现这个数据类型，是将其实现为一个`int`、`char`或者更复杂的类型都取决于你的实现。这个实现生成的库需要在使用了这个库的数据结构生成时进行链接。

另外还包括一个定义了一系列状态常量的头文件，用于作为某些函数的返回值类型。
