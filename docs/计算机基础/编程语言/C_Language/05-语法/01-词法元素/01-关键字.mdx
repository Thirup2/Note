**C 语言关键字如下所示**：

```
auto		double		int			struct
break		else		long		switch
case		enum		register	typedef
char		extern		return		union
const		for			signed		void
default		goto		sizeof		volatile
do			if			static		while
```



**关键字的意义**：

以上所列出的所有词法元素（全都是小写）都作为 C 语言关键字被保留，其中大部分是已经有实际意义的关键字，比如`double`、`int`等等。还有一部分是还未赋予其意义但仍被保留的关键字，这部分关键字可能在将来会被用上于是将其保留。

所有的关键字除了将其用于已指定意义的用途之外不能用于其他用途，例如：`int`只能用于表示类型而不能用作变量名。
