## 一、内容

```
++		// 后置递增
--		// 后置递减

++		// 前置递增
--		// 前置递减
```



## 二、基本属性

**操作**：前置递增递减运算符将运算对象`+1`（或`-1`），然后将改变后的对象作为求值结果进行返回；而后置递增递减运算符也将运算对象`+1`（或`-1`），但返回改变之前的值的副本。

**值类型**：前置和后置的递增递减运算符的运算对象都必须是可修改的左值，但前置运算符的求值结果是一个左值，而后置运算符的求值结果是一个右值。



## 三、注意事项

- 递增递减运算符会改变变量本身的值，如果在表达式中使用递增递减运算符，需要注意不要在同一个表达式中再次使用，否则很有可能导致未定义行为。
