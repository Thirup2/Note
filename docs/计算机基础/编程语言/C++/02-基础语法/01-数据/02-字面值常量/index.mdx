## 一、布尔类型

**字面值**：`true`/`false`



## 二、字符及字符串

**字面值**：

- 单引号中的单个字符为**字符字面值**，如 `'a'`，单引号中的字符可以是以下类型：

  - 任何可直接出现在程序中而不会引发歧义的明文字符

  - 转义序列

    | 含义       | 转义序列 |      | 含义       | 转义序列 |      | 含义           | 转义序列 |
    | ---------- | -------- | ---- | ---------- | -------- | ---- | -------------- | -------- |
    | 换行符     | `\n`     |      | 横向制表符 | `\t`     |      | 报警（响铃）符 | `\a`     |
    | 纵向制表符 | `\v`     |      | 退格符     | `\b`     |      | 双引号         | `\"`     |
    | 反斜线     | `\\`     |      | 问号       | `\?`     |      | 单引号         | `\'`     |
    | 回车符     | `\r`     |      | 进纸符     | `\f`     |      |                |          |

  - 字符编码

    - 对于 ASCII 码，可以使用`\ooo`的格式，其中`ooo`是一个八进制的数字，`\ooo`表示 ASCII 码中该编码对应的字符。`ooo`的范围是`0-337`（十进制的`0-255`），超过该范围的数字编译器可能会报错。最常用ASCII 码是空字符`\0`
    - 如果使用多字节或 Unicode 编码，可以使用`\xhhh`的格式，其中`hhh`是一个十六进制的数字，可以是任意位数

- 双引号中的单个或多个字符为**字符串字面值**，如`"string!"`。字符串中的字符可以是任意字符字面值。字符串字面值是一个以`'\0'`字符结尾的字符数组。

**字面值类型**：

- 默认：`char`或`char[]`
- `L`前缀：`wchar_t`或`wchar_t[]`
- `u`前缀：`char16_t`或`char16_t[]`
- `U`前缀：`char32_t`或`char32_t[]`
- `u8`前缀（仅用于字符串字面值）：`char[]`



## 三、整数类型

**字面值**：

- 十进制：任何十进制整数
- 八进制：以`0`开头的整数，任何位上的数字不得超过 7
- 十六进制：以`0x`或`0X`开头的整数，`10-15`分别用`a/A、b/B、c/C、d/D、e/E、f/F`表示

**字面值类型**：

- 默认
  - 十进制整数：带符号整数，最小是`int`。具体是`int`、`long`、`long long`中能容纳下该值的最小类型。
  - 八进制、十六进制整数：不一定是带符号整数，最小是`int`。具体是`int`、`unsigned int`、`long`、`unsigned long`、`long long`、`unsigned long long`中能容纳下该值的最小类型
- `u`/`U`后缀：无符号整数，最小是`unsigned`。具体是`unsigned`、`unsigned long`、`unsigned long long`中能容纳下该值的最小类型
- `l`/`L`后缀
  - 十进制整数：带符号整数，最小是`long`。具体是`long`、`long long`中能容纳下该值的最小类型
  - 八进制、十六进制整数：不一定是带符号整数，最小是`long`。具体是`long`、`unsigned long`、`long long`、`unsigned long long`中能容纳下该值的最小类型
- `ll/LL`后缀
  - 十进制整数：带符号整数，`long long`类型
  - 八进制、十六进制整数：最小为`long long`。具体是`long long`、`unsigned long long`中能容纳下该值的最小类型

> `u`/`U`后缀可以和`l`/`L`后缀或`ll`/`LL`后缀连用



## 四、浮点类型

**字面值**：

- 带小数点的数字：`3.14`、`3.`、`.14`
- 科学计数法：`31.4E-1`、`0.314e1`（`e`或`E`前是小数部分，之后的数字表示以 10 为底的指数）

**字面值类型**：

- 默认：都是`double`类型
- `f`或`F`后缀：`float`类型
- `l`或`L`后缀：`long `



## 五、指针类型

**字面值**：

- 空指针：`nullptr`

**字面值类型**：

- `nullptr`：`void*`（可赋值给任意类型指针变量）
