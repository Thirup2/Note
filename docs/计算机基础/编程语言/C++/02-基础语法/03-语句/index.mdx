---
title: 语句
---

C++ 语言中的大多数语句都以分号结束，比如一个表达式`ival + 5`，末尾加上分号就变成了**表达式语句**。

表达式语句的作用是执行表达式并丢弃掉求值结果。

### 空语句

最简单的语句是**空语句**，空语句中只含有一个单独的分号：

```cpp
;
```

空语句的最大的用途是在程序某处语法上需要一条语句，但逻辑上并不需要，可以使用空语句，例如我们希望输入流循环读取内容直到换行符为止：

```cpp
while(cin >> c && c !='\n')
    ;
```

此处用到了`while`语句，它由`while+(条件表达式)+处理语句`构成，其中处理语句在语法上是必须的，但在上面的例子中，我们并不需要一个处理语句，所以用空语句代替。



### 复合语句（块）

一条简单的表达式语句用一个分号进行分隔，在此基础上，我们也可以将多个简单表达式语句合在一起变成一条语句，就简单地用花括号将这些语句包裹起来即可。

```cpp
{
    string str;
    cin >> str;
    cout << str << endl;
}
```

由于有花括号`{}`的存在，以上三条语句在语法上被当作一条语句（只在块之外被当作一条语句），但在逻辑上却还是三条语句。

这在某些语法上只需要一条语句，但逻辑上需要多条语句的地方有用：

```cpp
while(val <= 10) {
    sum += val;
    ++val;
}
```

还是以`while`语句为例，由于语法上`while`只需要一条处理语句，所以用花括号将两条处理语句合成语法上的一条语句。
