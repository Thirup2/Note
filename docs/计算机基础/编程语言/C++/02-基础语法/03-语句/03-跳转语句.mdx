## 一、break 语句

```cpp
break;
```

**控制流**：`break`语句负责终止离它最近的`while`、`do while`、`for`或`switch`。并从这些语句之后的第一条语句开始继续执行。



## 二、continue 语句

```cpp
continue;
```

**控制流**：`continue`终止最近的循环（`while`、`for`、范围`for`、`do while`）中的当前迭代，并立即开始下一次迭代。

- 对于`while`和`do while`来说，下一步是判断条件的值。
- 对于`for`语句来说，下一步是执行`for`语句头的第三部分`expression`
- 对于范围`for`来说，则是用序列中的下一个元素初始化循环控制变量



## 三、return 语句

```cpp
return;
return expr;
```

**控制流**：`return`语句用来终止当前函数，并将值（如果有的话）返回到调用该函数的地方。



## 四、goto 语句

```cpp
goto label;
```

**控制流**：`goto`语句将控制流移动到`label`标签所在的位置，前提是该标签在`goto`语句的同一函数中。

标签的写法如下：

```cpp
end: return;
```

当我们使用`goto end;`这条语句时，控制流能直接到达`return;`语句。

:::caution 变量的定义

和`switch`类似，如果一个初始化语句有被跳过的可能，那么将会出错。

而如果是从后往前跳转而导致跳过了一个初始化语句，那么这是合法的，这意味着这个被初始化的对象将被销毁然后被重新创建。

:::
