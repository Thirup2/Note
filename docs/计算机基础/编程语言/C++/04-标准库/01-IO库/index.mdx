## 一、简介

**库**：`iostream`

**类型**：

- 输入流：`std::istream`
- 输出流：`std::ostream`

**对象**：

- 标准输入：`std::cin`（`istream`类型）
- 标准输出：`std::cout`（`ostream`类型）
- 标准错误：`std::cerr`（`ostream`类型）
- 标准日志：`std::clog`（`ostream`类型）

**操作**：

- 输入操作：`>>`（输入运算符，左侧对象为输入流对象，右侧对象为待输入对象，返回左侧对象。数据从输入端流向输入流对象再流向待输入对象）
- 输出操作：`<<`（输出运算符，左侧对象为输出流对象，右侧对象为待输出对象，返回左侧对象。数据从待输出对象流向输出流对象再流向输出端）

**特殊操作符**：

- `std::endl`：一个用在输出表达式中的操作符，当输出该操作符时，输出操作为换行，并刷新缓冲区
