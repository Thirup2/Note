---
title: 标准库
---

在 C++ 中，标准库可大致分为两种，一种是普通的标准库，一种是 STL（标准模板库）。

其中最复杂的部分就是 STL，完全可以将其当做 C++ 中的一种子语言进行学习，STL 主要包括以下内容：

- **容器**（Container）：各种数据结构，如`Vector`、`List`、`Deque`、`Set`、`Map`等，用来存放数据，STL 容器是一种类模板。
- **适配器**（Adapter）：一种用来修饰容器、仿函数或迭代器接口的东西。例如 STL 提供的 Queue 和 Stack，虽然看似是一种容器，但只能算是一种容器适配器，因为它们的底部完全借助`Deque`。改变函数接口的，称为函数适配器；改变容器接口的，称为容器适配器；改变迭代器接口的，称为迭代器适配器。
- **算法**（Algorithm）：STL 算法是一批函数模板，提供各种算法，例如`Sort`、`Search`、`Copy`、`Erase`等
- **迭代器**（Iterator）：容器和算法之间的胶合剂，即所谓的“泛型指针”，共有五种类型，以及其他衍生变化。所有 STL 容器都附带有自己专属的迭代器
- **函数对象**（Function object）：行为类似函数，可作为算法的某种策略。函数对象是一种重载了`Operator()`的类或类模板。
- **分配器**（Allocator）：负责空间配置与管理。分配器是一个实现了动态空间配置、空间管理、空间释放的类模板
