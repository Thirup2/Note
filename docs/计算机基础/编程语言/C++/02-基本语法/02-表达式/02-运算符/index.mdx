---
title: 运算符
---

本节将按照相同优先级为一组，不同优先级从高到低的顺序列出所有运算符：

:::info 关于括号

实际上在所有运算符的优先级之上还有一个括号运算符，其优先于其他所有运算符；

另外，括号可以嵌套，内层括号优先于外层括号。

所以，如果你不确定要使用的运算符的具体优先级，可以用括号来调控。

:::

- 第 1 组（最高优先级）

  | 运算符 | 功能           | 用法              | 结合律 | 求值顺序 |
  | ------ | -------------- | ----------------- | ------ | -------- |
  | `::`   | 全局作用域     | `::name`          | 左     | NULL     |
  | `::`   | 类作用域       | `class::name`     | 左     | NULL     |
  | `::`   | 命名空间作用域 | `namespace::name` | 左     | NULL     |

- 第 2 组

  | 运算符 | 功能     | 用法              | 结合律 | 求值顺序 |
  | ------ | -------- | ----------------- | ------ | -------- |
  | `.`    | 成员选择 | `object.member`   | 左     | NULL     |
  | `->`   | 成员选择 | `pointer->member` | 左     | NULL     |
  | `[]`   | 下标     | `expr[expr]`      | 左     | NULL     |
  | `()`   | 函数调用 | `name(expr_list)` | 左     | NULL     |
  | `()`   | 类型构造 | `type(expr_list)` | 左     | NULL     |

- 第 3 组

  | 运算符          | 功能         | 用法                    | 结合律 | 求值顺序 |
  | --------------- | ------------ | ----------------------- | ------ | -------- |
  | `++`            | 后置递增运算 | `lvalue++`              | 右     | NULL     |
  | `--`            | 后置递减运算 | `lvalue--`              | 右     | NULL     |
  | `typeid`        | 类型ID       | `typeid(type)`          | 右     | NULL     |
  | `typeid`        | 运行时类型ID | `typeid(type)`          | 右     | NULL     |
  | `explicit cast` | 类型转换     | `cast_name<type>(expr)` | 右     | NULL     |
  
- 第 4 组

  | 运算符      | 功能         | 用法              | 结合律 | 求值顺序 |
  | ----------- | ------------ | ----------------- | ------ | -------- |
  | `++`        | 前置递增运算 | `++lvalue`        | 右     | NULL     |
  | `--`        | 前置递减运算 | `--lvalue`        | 右     | NULL     |
  | `~`         | 位求反       | `~expr`           | 右     | NULL     |
  | `!`         | 逻辑非       | `!expr`           | 右     | NULL     |
  | `-`         | 一元负号     | `-expr`           | 右     | NULL     |
  | `+`         | 一元正号     | `+expr`           | 右     | NULL     |
  | `*`         | 解引用       | `*expr`           | 右     | NULL     |
  | `&`         | 取地址       | `&lvalue`         | 右     | NULL     |
  | `()`        | 类型转换     | `(type)expr`      | 右     | NULL     |
  | `sizeof`    | 对象的大小   | `sizeof expr`     | 右     | NULL     |
  | `sizeof`    | 类型的大小   | `sizeof(type)`    | 右     | NULL     |
  | `sizeof...` | 参数包的大小 | `sizeof...(name)` | 右     | NULL     |
  | `new`       | 创建对象     | `new type`        | 右     | NULL     |
  | `new[]`     | 创建数组     | `new type[size]`  | 右     | NULL     |
  | `delete`    | 释放对象     | `delete expr`     | 右     | NULL     |
  | `delete[]`  | 释放数组     | `delete[] expr`   | 右     | NULL     |
  | `noexcept`  | 能否抛出异常 | `noexcept(expr)`  | 右     | NULL     |

- 第 5 组

  | 运算符 | 功能               | 用法                  | 结合律 | 求值顺序 |
  | ------ | ------------------ | --------------------- | ------ | -------- |
  | `->*`  | 指向成员选择的指针 | `ptr->*ptr_to_member` | 左     | NULL     |
  | `.*`   | 指向成员选择的指针 | `obj.*ptr_to_member`  | 左     | NULL     |

- 第 6 组

  | 运算符 | 功能         | 用法        | 结合律 | 求值顺序 |
  | ------ | ------------ | ----------- | ------ | -------- |
  | `*`    | 乘法         | `expr*expr` | 左     | NULL     |
  | `/`    | 除法         | `expr/expr` | 左     | NULL     |
  | `%`    | 取模（取余） | `expr%expr` | 左     | NULL     |

- 第 7 组

  | 运算符 | 功能 | 用法        | 结合律 | 求值顺序 |
  | ------ | ---- | ----------- | ------ | -------- |
  | `+`    | 加法 | `expr+expr` | 左     | NULL     |
  | `-`    | 减法 | `expr-expr` | 左     | NULL     |

- 第 8 组

  | 运算符 | 功能     | 用法         | 结合律 | 求值顺序 |
  | ------ | -------- | ------------ | ------ | -------- |
  | `<<`   | 向左移位 | `expr<<expr` | 左     | NULL     |
  | `>>`   | 向右移位 | `expr>>expr` | 左     | NULL     |

- 第 9 组

  | 运算符 | 功能     | 用法         | 结合律 | 求值顺序 |
  | ------ | -------- | ------------ | ------ | -------- |
  | `<`    | 小于     | `expr<expr`  | 左     | NULL     |
  | `<=`   | 小于等于 | `expr<=expr` | 左     | NULL     |
  | `>`    | 大于     | `expr>expr`  | 左     | NULL     |
  | `>=`   | 大于等于 | `expr>=expr` | 左     | NULL     |

- 第 10 组

  | 运算符 | 功能   | 用法         | 结合律 | 求值顺序 |
  | ------ | ------ | ------------ | ------ | -------- |
  | `==`   | 相等   | `expr==expr` | 左     | NULL     |
  | `!=`   | 不相等 | `expr!=expr` | 左     | NULL     |

- 第 11 组

  | 运算符 | 功能 | 用法        | 结合律 | 求值顺序 |
  | ------ | ---- | ----------- | ------ | -------- |
  | `&`    | 位与 | `expr&expr` | 左     | NULL     |

- 第 12 组

  | 运算符 | 功能   | 用法        | 结合律 | 求值顺序 |
  | ------ | ------ | ----------- | ------ | -------- |
  | `^`    | 位异或 | `expr^expr` | 左     | NULL     |

- 第 13 组

  | 运算符              | 功能 | 用法                        | 结合律 | 求值顺序 |
  | ------------------- | ---- | --------------------------- | ------ | -------- |
  | <code>&#124;</code> | 位或 | <code>expr&#124;expr</code> | 左     | NULL     |

- 第 14 组

  | 运算符 | 功能   | 用法         | 结合律 | 求值顺序 |
  | ------ | ------ | ------------ | ------ | -------- |
  | `&&`   | 逻辑与 | `expr&&expr` | 左     | 先左后右 |

- 第 15 组

  | 运算符                    | 功能   | 用法                              | 结合律 | 求值顺序 |
  | ------------------------- | ------ | --------------------------------- | ------ | -------- |
  | <code>&#124;&#124;</code> | 逻辑或 | <code>expr&#124;&#124;expr</code> | 左     | 先左后右 |

- 第 16 组

  | 运算符 | 功能 | 用法             | 结合律 | 求值顺序                                             |
  | ------ | ---- | ---------------- | ------ | ---------------------------------------------------- |
  | `?:`   | 条件 | `expr?expr:expr` | 右     | 先左，<br />若左为真，则求中；<br />若左为假，则求右 |

- 第 17 组

  | 运算符 | 功能 | 用法          | 结合律 | 求值顺序 |
  | ------ | ---- | ------------- | ------ | -------- |
  | `=`    | 赋值 | `lvalue=expr` | 右     | NULL     |

- 第 18 组

  | 运算符                                                       | 功能     | 用法            | 结合律 | 求值顺序 |
  | ------------------------------------------------------------ | -------- | --------------- | ------ | -------- |
  | `*=`, `/=`, `%=`<br />`+=`, `-=`<br />`<<=`, `>>=`<br />`&=`, <code>&#124;=</code>, `^=` | 复合赋值 | `lvaue+=expr`等 | 右     | NULL     |

- 第 19 组

  | 运算符  | 功能     | 用法         | 结合律 | 求值顺序 |
  | ------- | -------- | ------------ | ------ | -------- |
  | `throw` | 抛出异常 | `throw expr` | 右     | NULL     |

- 第 20 组（最低优先级）

  | 运算符 | 功能 | 用法         | 结合律 | 求值顺序 |
  | ------ | ---- | ------------ | ------ | -------- |
  | `,`    | 逗号 | `expr, expr` | 左     | 先左后右 |
