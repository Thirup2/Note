## 一、运算符优先级表

本节将按照相同优先级为一组，不同优先级从高到低的顺序列出所有运算符：

- 第 1 组（最高优先级）

  | 运算符 | 功能           | 用法              | 结合律 | 求值顺序 | 接受值类型 | 结果值类型 |
  | ------ | -------------- | ----------------- | ------ | -------- | ---------- | ---------- |
  | `::`   | 全局作用域     | `::name`          | 左     | NULL     | NULL       | NULL       |
  | `::`   | 类作用域       | `class::name`     | 左     | NULL     | NULL       | NULL       |
  | `::`   | 命名空间作用域 | `namespace::name` | 左     | NULL     | NULL       | NULL       |

- 第 2 组

  | 运算符 | 功能     | 用法              | 结合律 | 求值顺序 | 接受值类型 | 结果值类型 |
  | ------ | -------- | ----------------- | ------ | -------- | ---------- | ---------- |
  | `.`    | 成员选择 | `object.member`   | 左     | NULL     | NULL       | NULL       |
  | `->`   | 成员选择 | `pointer->member` | 左     | NULL     | NULL       | NULL       |
  | `[]`   | 下标     | `expr[expr]`      | 左     | NULL     | NULL       | NULL       |
  | `()`   | 函数调用 | `name(expr_list)` | 左     | NULL     | NULL       | NULL       |
  | `()`   | 类型构造 | `type(expr_list)` | 左     | NULL     | NULL       | NULL       |

- 第 3 组

  | 运算符 | 功能 | 用法 | 结合律 | 求值顺序 | 接受值类型 | 结果值类型 |
  | ------ | ---- | ---- | ------ | -------- | ---------- | ---------- |
  |        |      |      |        |          |            |            |
