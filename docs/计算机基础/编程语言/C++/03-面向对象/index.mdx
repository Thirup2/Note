---
title: 面向对象
---

**面向对象程序设计**的核心思想是**数据抽象**、**继承**和**动态绑定**。

通过使用数据抽象，我们可以将类的接口与实现相分离；

使用继承，可以定义相似的类型并对其相似关系建模；

使用动态绑定，可以在一定程度上忽略相似类型的区别，而以统一的方式使用它们的对象。



## 关键字总结

1. `const`（成员函数）：所有的声明和定义中。参数列表后，函数体前。
2. `inline`：所有定义中，声明中可选。返回类型前。
3. `mutable`：数据成员声明类型前
4. `explicit`：类内声明中。构造函数名前
5. `static`：类内声明中。数据成员类型和成员函数返回类型前。
