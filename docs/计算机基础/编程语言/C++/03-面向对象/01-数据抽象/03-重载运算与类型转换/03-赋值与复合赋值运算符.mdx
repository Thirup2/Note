## 一、赋值运算符

> 赋值运算符中比较重要的有三个：[拷贝赋值运算符](../../构造与拷贝控制/拷贝构造与赋值/拷贝赋值运算符/)、[移动赋值运算符](../../构造与拷贝控制/移动构造与赋值/移动操作/#move_assign)以及[列表赋值运算符](../../构造与拷贝控制/构造函数/列表构造与赋值/#list_assign)，已经在“构造与拷贝控制”进行了说明，此处不再重复。

除此之外，类还可以定义其他赋值运算符以使用别的类型作为右侧运算对象，只要遵循如下的规则：

- 函数类型：成员函数
- 参数：一个用于赋值的运算对象
- 返回值：左侧运算对象的引用

另外，对于拷贝和移动赋值运算符之外的其他赋值运算符，基本上都不用考虑自赋值的情况。



## 二、复合赋值运算符

- 函数类型：常为成员函数
- 参数：如果定义为成员函数，则只需要一个用于复合赋值的运算对象
- 返回值：左侧运算对象的引用

**示例**：

```cpp
Sales_data& Sales_data::operator+=(const Sales_data &rhs)
{
    units_sold += rhs.units_sold;
    revenue += rhs.revenue;
    return *this;
}
```
