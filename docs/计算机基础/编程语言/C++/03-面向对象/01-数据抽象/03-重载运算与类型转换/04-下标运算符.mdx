表示容器的类通常可以通过元素在容器中的位置访问元素，这些类一般会定义下标运算符`operator[]`。该函数基本要求如下：

- 函数类型：成员函数
- 参数：一个下标类型的值（下标类型根据具体的类确定，例如`map`的下标为关键字类型，其余大部分类型的下标为一个整数）
- 返回值：所访问元素的引用

**示例**：

```cpp
class StrVec {
public:
    std::string& operator[](std::size_t n) { return elements[n]; }
    const std::string& operator[](std::size_t n) const { return elements[n]; }
private:
    std::string *elements;
};
```

:::info 下标运算符的重载

下标运算符通常需要同时定义常量版本和非常量版本。

目的在于当作用于一个常量对象时，下标运算符返回常量引用以确保我们不会给返回的对象赋值。

:::

另外，内置类型的下标运算符并不检查下标越界的情况，所以通常重载的下标运算符也并不检查下标越界的情况。
