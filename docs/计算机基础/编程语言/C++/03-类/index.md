## 一、简介

**类**：即一种可定义的数据结构，包括数据类型和相关操作在内的一种结构，以关键字`class`或`struct`开始，基本格式如下：

```cpp
struct ClassName {
    ...
};
```

**类类型**：自定义类所代表的类型

**类成员**：在类的花括号之间的成员就是类成员，包括数据成员和成员函数

**数据成员**：类的数据成员定义了类的对象的具体内容。可以为数据成员提供一个**类内初始值**。

**成员函数**：成员函数是定义为类的一部分的函数，有时也被成为**方法**
