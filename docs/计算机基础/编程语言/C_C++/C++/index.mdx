## 简介

此文档是关于 C++ 编程语言基础的总结笔记，也是一个知识手册。主要是关于 C++ 基本语法的总结，虽然说是基础，但是本文档也是比较全面的，也包含了标准库、类以及模板的内容。

本文档将从 4 个方面对 C++ 进行总结，首先是基础部分，基本包括数据、表达式、语句、函数这 4 个方面，这也算是每一门编程语言必有的 4 个语法要素。其次是 C++ 类的相关内容，算是 C++ 面向对象特性里的一个非常重要的组成部分。第三是 C++ 标准库，大部分语言都会有自己的标准库，而 C++ 的标准库则是出了名的复杂。最后一部分就基本上是模板与泛型编程相关的内容了，模板的使用可以大大提高类的可重用性，同时也是一个比较难的部分。



## C++ 标准

在本文档开始编写时，C++ 的 ISO 标准还仅仅只是 C++20，而实际上有一部分编译器支持的标准已经到 C++23 了。

本文档在编写过程中将以 ISO 标准为准，而由于 C++11 较之前的标准变动较大，故以 C++11 的 ISO 标准为基本标准，后续的所有标准中更新了的内容将额外给出（具体样式如下）。也就是说，你可以在本文档中看到大部分不是特别列出的的 “C++ 就是……”、“C++ 可以……” 等语句都应该转换为 “C++11 就是……”、“C++11 可以……” 等等。这表示如果你使用 C++11 之前的 C++ 版本或许将与本文档内容产生冲突。
