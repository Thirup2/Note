## 一、元素与标签

在 HTML 文档中，内容是由元素构成的，而元素则由标签进行标识。

分为双标签和单标签，例如：

- `<p>这是一个双标签</p>`：其中`p`元素则由`<p>`和`</p>`两个标签组成
- `<br>`：这就是一个单标签元素，该元素可以实现换行功能。单标签也可以写成`<br/>`以及`<br />`的形式



## 二、文档基本结构

一个最简单的文档也应该包含以下内容：

```html
<!DOCTYPE html>
<html lang="en">
    
	<head>
    	<title>Title</title>
	</head>
    
	<body>
	</body>
    
</html>
```

- `<!DOCTYPE html>`：这一行用来声明文档类型是 HTML，以及其版本为 HTML5（因为只有这个版本能这样声明）
- `<html lang="en">...</html>`：这一对标签表示文档的开头和结尾，所有的内容都应该在这对标签内。其中需要用`lang`属性声明文档使用的语言。
- `<head>...</head>`：这一对标签用于添加说明文档本身的信息，其中至少有一个`title`元素
- `<title>...</title>`：这一对标签用于添加文档的标题，会显示在页面标签栏上
- `<body>...</body>`：这一对标签中的内容即网页所显示的内容，几乎所有内容都应该写在这对标签中

:::info 关于乱码

如果只包含以上基本结构在浏览器中渲染你的文档时，如果你的内容出现了乱码情况，请在`<head></head>`部分添加以下内容：

```HTML
<meta charset="UTF-8">
```

其中`UTF-8`可以是其他编码，但需要和你的源文件采用的编码一致。

:::
