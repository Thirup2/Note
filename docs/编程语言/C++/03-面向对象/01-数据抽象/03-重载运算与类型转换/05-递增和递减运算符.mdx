## 一、前置递增递减运算符

- 函数类型：成员函数
- 参数：无
- 返回值：递增对象并返回递增后对象的引用

**示例**：

```cpp
class StrBlobPtr {
public:
    StrBlobPtr& operator++();
    StrBlobPtr& operator--();
};
```



## 二、后置递增递减运算符

- 函数类型：成员函数

- 参数：一个不被使用的`int`形参

  > 之所以需要这个形参，是因为如果不使用这样的方法就无法区分`++`和`--`运算符到底是前置版本还是后置版本。所以通过添加一个形参的方法来标识后置版本。
  >
  > 从语法上来说这个形参可以使用，但在实际过程中通常不会这么做，因为当我们使用运算符形式调用时无法传入这个形参，只有通过函数调用的形式才能传入这个形参，所以这个形参的用途通常只用来标识递增递减运算符的后置版本。

- 返回值：递增对象并返回递增前对象的拷贝

**示例**：

```cpp
class StrBlobPtr {
public:
    StrBlobPtr operator++(int);
    StrBlobPtr operator--(int);
};
```

:::info 后置版本的调用

上面定义的函数首先可以按照运算符的形式调用：

```cpp
StrBlobPtr sbt;
sbt++;
```

也可以按照函数形式调用，但为了和前置版本分别，函数形式调用时也需要传递一个值：

```cpp
sbt.operator++(0);		// 后置版本
sbt.operator++();		// 前置版本
```

由于后置版本在实现时通常并不会使用这个形参，所以当我们使用函数形式调用后置版本递增递减运算符时，通常也不需要考虑具体传入的`int`类型值是多少。

:::
