## 一、while 语句

```cpp
while (condition)
    statement;
```

**控制流**：计算`condition`的值，当其为`true`时，执行`statement;`，然后再重复上述过程；当其为`false`时，跳过`while`语句。



## 二、for 语句

```cpp
for (init-statement; condition; expression)
    statement;
```

**控制流**：执行`init-statement`，计算`condition`，当其为`true`时，执行`statement;`，然后执行`expression`，然后再重复除了`init-statement`之外的上述过程；当`condition`的计算值为`false`时，跳过`for`语句。

`init-statement`要求是一条语句，必须是以下三种形式中的一种：声明语句、表达式语句或空语句。

`expression`是表达式，也就是说可以使用`,`运算符并列多个表达式。

:::info 省略

`for`语句中的`init-statement`、`condition`、`expression`都可以省略（即用空语句代替）。

省略`init-statement`和`expression`时比较好理解。

而如果省略`condition`，那么默认`condition`的值为`true`。

:::



## 三、范围 for 语句

```cpp
for (declaration : expression)
    statement;
```

**控制流**：`expression`必须是一个**初始值列表**、**数组**或定义了能返回迭代器的`begin`和`end`成员的**类**。`declaration`定义一个变量，`expression`序列中的每个元素必须是该类型或者能够转换成该类型。范围`for`语句将`declaration`中定义的变量从`expression`的`begin`指向的元素开始，一直移动到`expression`返回的`end`指向的元素后一个位置，此时将退出这个范围`for`语句。`declaration`中定义的变量每一次改变值，都将执行一次`statement;`。

需要注意的是，如果想要通过范围`for`语句修改序列中的元素值，那么需要将`declaration`中定义的变量定义成对序列中元素类型的引用。

范围`for`语句等价于：

```cpp
for (auto beg = v.begin(), end = v.end(); beg != end; ++beg) {
    auto &r = *beg;
    statement;
}
```



## 四、do while 语句

```cpp
do
    statement;
while (condition);
```

**控制流**：首先执行`statement;`，然后按照和普通`while`语句相同的流程执行。需要注意的是在`condition`之后不要忘记添加分号表示语句的结束。

需要注意的是：在普通`while`语句中，我们可以在`condition`的部分定义变量，但由于`do while`语句需要先执行`statement;`，所以不能再在`condition`中定义变量供`statement;`使用。
