---
title: 数据
---

C++ 基础语法部分的第一个部分是**数据**。

在 C++ 中有各种各样的类型，整数类型、字符类型、指针类型、引用类型等等，每种不同的类型都有不尽相同的用法。

然而在硬件层面，一份数据就只是内存中的一串 bit 序列，在 C++ 中，有很多数据的尺寸是相同的，也就是说这些数据在内存中是同样长的一串 bit 序列，而计算机怎样才能知道该如何处理这些看起来相同但实际有区别的数据呢？其实计算机执行的计算只有两种：整数运算、浮点运算。在这一层面上，我们可以将数据类型大致分为两类即**整型**和**浮点型**。不论是整数类型还是字符类型，它们在 C++ 的语法中虽然看上去进行的是不同的操作，但实际上在底层执行的都是整型运算，是同一套运算法则。这是 C++ 对于 硬件而言的数据。

而 C++ 对于用户而言的数据则可以分得更加细致，表示真假值的布尔类型、表示整数的各种尺寸的整数类型、表示字符的字符类型等等。通过编程语言对数据的抽象，我们可以使用各种各样的类型来表示各种复杂的数据，尽管某些类型在机器中实际上是没有区别的。

在实际编程中，我们最好本分地使用编程语言为我们提供的抽象出来的各种数据类型，而不要试图用一个整数类型来保存字符或用字符类型保存整数等。了解其底层实现只是为了我们更好的理解 C++ 中的各种现象而并非鼓励我们去张冠李戴。
