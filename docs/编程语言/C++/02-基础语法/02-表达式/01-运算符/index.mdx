---
title: 运算符
---

## 优先级 1（最高）

| 运算符 | 功能         | 用法                                                         | 结合律 |
| :----- | ------------ | ------------------------------------------------------------ | ------ |
| `::`   | 作用域运算符 | 1. 全局作用域：`::name`<br />2. 类作用域：`class::name`<br />3. 命名空间作用域：`namespace::name` | 左     |



## 优先级 2

| 运算符 | 功能     | 用法              | 结合律 |
| :----- | -------- | ----------------- | ------ |
| `.`    | 成员选择 | `object.member`   | 左     |
| `->`   | 成员选择 | `pointer->member` | 左     |
| `[]`   | 下标     | `expr[expr]`      | 左     |
| `()`   | 函数调用 | `name(expr_list)` | 左     |
| `()`   | 类型构造 | `type(expr_list)` | 左     |



## 优先级 3

| 运算符          | 功能         | 用法                                                         | 结合律 |
| --------------- | ------------ | ------------------------------------------------------------ | ------ |
| `++`            | 后置递增运算 | `lvalue++`                                                   | 右     |
| `--`            | 后置递减运算 | `lvalue--`                                                   | 右     |
| `typeid`        | 类型 ID      | 1. 类型 ID：`typeid(type)`<br />2. 运行时类型 ID：`typeid(expr)` | 右     |
| `explicit cast` | 类型转换     | `cast_name<type>(expr)`                                      | 右     |



## 优先级 4

| 运算符     | 功能         | 用法                                                         | 结合律 |
| ---------- | ------------ | ------------------------------------------------------------ | ------ |
| `++`       | 前置递增运算 | `++lvalue`                                                   | 右     |
| `--`       | 前置递减运算 | `--lvalue`                                                   | 右     |
| `~`        | 位求反       | `~expr`                                                      | 右     |
| `!`        | 逻辑非       | `!expr`                                                      | 右     |
| `-`        | 一元负号     | `-expr`                                                      | 右     |
| `+`        | 一元正号     | `+expr`                                                      | 右     |
| `*`        | 解引用       | `*expr`                                                      | 右     |
| `&`        | 取地址       | `&lvalue`                                                    | 右     |
| `()`       | 类型转换     | `(type)expr`                                                 | 右     |
| `sizeof`   | 大小         | 1. 对象的大小：`sizeof expr`<br />2. 类型的大小：`sizeof(type)`<br />3. 参数包的大小：`sizeof...(name)` | 右     |
| `new`      | 创建对象     | `new type`                                                   | 右     |
| `new[]`    | 创建数组     | `new type[size]`                                             | 右     |
| `delete`   | 释放对象     | `delete expr`                                                | 右     |
| `delete[]` | 释放数组     | `delete[] expr`                                              | 右     |
| `noexcept` | 能否抛出异常 | `noexcept(expr)`                                             | 右     |



## 优先级 5

| 运算符 | 功能               | 用法                  | 结合律 |
| ------ | ------------------ | --------------------- | ------ |
| `->*`  | 指向成员选择的指针 | `ptr->*ptr_to_member` | 左     |
| `.*`   | 指向成员选择的指针 | `obj.*ptr_to_member`  | 左     |



## 优先级 6

| 运算符 | 功能         | 用法          | 结合律 |
| ------ | ------------ | ------------- | ------ |
| `*`    | 乘法         | `expr * expr` | 左     |
| `/`    | 除法         | `expr / expr` | 左     |
| `%`    | 求模（取余） | `expr % expr` | 左     |



## 优先级 7

| 运算符 | 功能 | 用法          | 结合律 |
| ------ | ---- | ------------- | ------ |
| `+`    | 加法 | `expr + expr` | 左     |
| `-`    | 减法 | `expr - expr` | 左     |



## 优先级 8

| 运算符 | 功能   | 用法           | 结合律 |
| ------ | ------ | -------------- | ------ |
| `<<`   | 左移位 | `expr << expr` | 左     |
| `>>`   | 右移位 | `expr >> expr` | 左     |



## 优先级 9

| 运算符 | 功能     | 用法           | 结合律 |
| ------ | -------- | -------------- | ------ |
| `<`    | 小于     | `expr < expr`  | 左     |
| `<=`   | 小于等于 | `expr <= expr` | 左     |
| `>`    | 大于     | `expr > expr`  | 左     |
| `>=`   | 大于等于 | `expr >= expr` | 左     |



## 优先级 10

| 运算符 | 功能   | 用法           | 结合律 |
| ------ | ------ | -------------- | ------ |
| `==`   | 相等   | `expr == expr` | 左     |
| `!=`   | 不相等 | `expr != expr` | 左     |



## 优先级 11

| 运算符 | 功能 | 用法          | 结合律 |
| ------ | ---- | ------------- | ------ |
| `&`    | 位与 | `expr & expr` | 左     |



## 优先级 12

| 运算符 | 功能   | 用法          | 结合律 |
| ------ | ------ | ------------- | ------ |
| `^`    | 位异或 | `expr ^ expr` | 左     |



## 优先级 13

| 运算符              | 功能 | 用法                          | 结合律 |
| ------------------- | ---- | ----------------------------- | ------ |
| <code>&#124;</code> | 位或 | <code>expr &#124; expr</code> | 左     |



## 优先级 14

| 运算符 | 功能   | 用法           | 结合律 | 求值顺序                        |
| ------ | ------ | -------------- | ------ | ------------------------------- |
| `&&`   | 逻辑与 | `expr && expr` | 左     | 先左后右，左为`false`则不用求右 |



## 优先级 15

| 运算符                    | 功能   | 用法                                | 结合律 | 求值顺序                       |
| ------------------------- | ------ | ----------------------------------- | ------ | ------------------------------ |
| <code>&#124;&#124;</code> | 逻辑或 | <code>expr &#124;&#124; expr</code> | 左     | 先左后右，左为`true`则不用求右 |



## 优先级 16

| 运算符 | 功能 | 用法                 | 结合律 | 求值顺序                                                     |
| ------ | ---- | -------------------- | ------ | ------------------------------------------------------------ |
| `?:`   | 条件 | `expr ? expr : expr` | 右     | 先 1 后 2 或 3<br />1 为`true`则求 2 不求 3<br />1 为`false`则求 3 不求 2 |



## 优先级 17

| 运算符                                                       | 功能 | 用法                                    | 结合律 |
| ------------------------------------------------------------ | ---- | --------------------------------------- | ------ |
| `=`<br />`*=`、`/=`、`%=`<br />`+=`、`-=`<br />`<<=`、`>>=`<br />`&=`、<code>&#124;=</code>、`^=` | 赋值 | `lvalue = expr`<br />`lvalue += expr`等 | 右     |



## 优先级 18

| 运算符  | 功能     | 用法         | 结合律 |
| ------- | -------- | ------------ | ------ |
| `throw` | 抛出异常 | `throw expr` | 右     |



## 优先级 19（最低）

| 运算符 | 功能 | 用法         | 结合律 | 求值顺序 |
| ------ | ---- | ------------ | ------ | -------- |
| `,`    | 逗号 | `expr, expr` | 左     | 先左后右 |
