## 一、内容

```
!		// 逻辑非

<		// 小于关系
<=		// 小于等于关系
>		// 大于关系
>=		// 大于等于关系

==		// 相等关系
!=		// 不等关系

&&		// 逻辑与

||		// 逻辑或
```



## 二、基本属性

**值类型**：逻辑运算符的运算对象都是`bool`类型值或能够转换为`bool`类型值的其他类型，而关系运算符的运算对象则是能够进行关系运算的值，这些运算返回的结果都是`bool`类型值。这些运算符的运算对象以及返回值都是右值。

**结合律**：除了逻辑非运算符`!`是右结合律之外，其余运算符都是左结合律

**求值顺序**：`&&`和`||`有求值顺序。它们的求值顺序遵循**短路求值**策略，`&&`当二者都为真时才返回真，`||`当二者都为假时才返回假。所以`&&`先对左运算对象求值，为真时才对右运算对象求值；`||`先对左运算对象求值，为假时才对右运算对象求值。



## 三、注意事项

- 关系运算符的运算结果是一个`bool`值，所以需要注意并列多个关系运算符的表达式很容易出错

  ```cpp
  if (i < j < k);
  ```

  根据结合律，其中`i < j`先计算，其值非 0 即 1。如果要判断`j`是否在`i~k`之间，应该写成

  ```cpp
  if (j > i && j < k)
  ```

- 用布尔字面值进行关系运算会很容易出错，例如

  ```cpp
  if (val == true);
  ```

  如果`val`是一个布尔值，那这个表达式可以直接简化为`if (val)`；而如果`val`不是一个布尔值，那上述表达式可替换为`if (val == 1)`。然而实际上这个语句的意图可能是当`val`为`true`时进入分支，而不是当`val`为 1 时进入分支。`val == true`这个表达式中进行的转换不是`val`转换为`bool`类型值，而是`true`字面值转换为`int`类型值。
