## 一、内容

```
~		// 位求反

<<		// 左移位
>>		// 右移位

&		// 位与

^		// 位异或

|		// 位或
```



## 二、基本属性

**值类型**：位运算符的运算对象都是整数类型，以上运算符都接受右值运算对象，并返回右值结果。

**结合律**：以上运算符都遵循左结合律



## 三、功能

位运算符将一个整数看成是二进制位的集合，并在位的尺度对数据进行操作。位运算符的运算对象的类型将进行自动的整型提升，如果小于`int`的整型（包括`short`等整数类型）的对象所保存的值可以存在`int`里，就提升成`int`，否则就提升成`unsigned`。对于大于`int`的其他整型（如`char32_t`，不包括`long`等整数类型）提升成`int`、`unsigned`、`long`、`unsigned long`、`long long`和`unsigned long long`中最小的一种类型，前提是转换后的类型要能够容纳原类型所有可能的值。

- 移位运算符：左侧运算对象是被移位的对象。右侧运算对象不能为负，且其值必须严格小于结果的位数，否则会发生未定义行为。其求值结果是左侧运算对象经过移位操作后的值。
  - 左移运算符（`<<`）：将当前的序列往左移动右侧运算对象指定的位数，左侧移出序列范围的位将被舍弃，右侧的空位用 0 补充
  - 右移运算符（`>>`）：将当前的序列往右移动右侧运算对象指定的位数，右侧移出序列范围的位将被舍弃。如果运算对象是无符号类型，则左侧的空位用 0 补充；如果运算对象是带符号类型，左侧空位用符号位的副本补充或 0 补充，如何选择根据具体环境而定。
- 位运算符：接受 1 个或 2 个运算对象，按照位运算符规定的运算得出结果并返回
  - 位求反（`~`）：接受 1 个右侧运算对象，将其按位进行求反（0 变 1，1 变 0）并返回结果
  - 位与（`&`）：将两个运算对象按照整型提升成相同类型，然后按位依次对应，当两个位都为 1 时返回 1，否则返回 0。按照此原则对每一对位组合进行运算，将每一对位运算得出的结果依次放入一个新对象中，最终返回这些位组合出来的新值。
  - 位或（`|`）：将两个运算对象按照整型提升成相同类型，然后按位依次对应，当两个位都是 0 时返回 0，否则返回 1。按照此原则对每一对位组合进行运算，将每一对位运算得出的结果依次放入一个新对象中，最终返回这些位组合出来的新值。
  - 位异或（`^`）：将两个运算对象按照整型提升成相同类型，然后按位依次对应，当两个位不同时返回 1，否则返回 0。按照此原则对每一对位组合进行运算，将每一对位运算得出的结果依次放入一个新对象中，最终返回这些位组合出来的新值。
