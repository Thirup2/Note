## 一、内容

```
=
*=	/=	%=
+=	-=
<<=	>>=
&=	|=	^=
```



## 二、基本属性

**值类型**：赋值运算符的左侧运算对象必须是一个可修改的左值。赋值运算的结果也是它的左侧运算对象，仍然是一个左值。结果的类型是左侧运算对象的类型，如果赋值运算符的左右两个运算对象类型不同，则右侧运算对象将转换为左侧运算对象的类型。

**结合律**：所有赋值运算符都满足右结合律



## 三、注意事项

- 复合赋值运算符等价于将左侧运算对象和右侧运算对象执行`=`前操作符之后得到的值赋值给左侧运算对象，例如`a += b`等价于`a = a + b`，但前者只需要对`a`求一次值，而后者需要对`a`求两次值。
