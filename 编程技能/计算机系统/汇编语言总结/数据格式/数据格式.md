# 一. 数据类型

| Intel 数据类型 | 大小（字节） | 汇编代码后缀 |
| -------------- | ------------ | ------------ |
| 字节           | 1            | `b`          |
| 字             | 2            | `w`          |
| 双字           | 4            | `l`          |
| 四字           | 8            | `q`          |
| 单精度         | 4            | `s`          |
| 双精度         | 8            | `l`          |



# 二. 使用

大多数 x86-64 的汇编指令除了指令本身外，还需要添加一个字符的后缀，这个后缀就是上表中的汇编代码后缀，用于表示指令中操作数的大小，如：数据传送指令有四个不同大小的变种：`movb`（传送字节）、`movw`（传送字）、`movl`（传送双字）、`movq`（传送四字）

**注意**：

- 操作数如果使用寄存器，那么使用的寄存器大小必须跟指令指定的相同
