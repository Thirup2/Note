# 一. 出现问题的代码及具体问题
```c++
...
vector num1;
vector num2;
int n;
while(cin >> n)
{
 num1.push_back(n);
}
while(cin >> n)
{
 num2.push_back(n);
}
...
```
在以上代码中, 第一个循环输入值正常, 但其停止条件是EOF, 但在终端里输入了`^z`之后, 整个程序就结束了, 或者说是整个程序的输入过程的结束了, 根本就不会进第二个循环.
