四. 其他指令

一. #error

#error text of error message
  • 当程序编译之后，#error指令允许生成错误信息

二. #line

#line number "string"
  • 它通知预处理器number是下一行输入的行号。如果给出了可选部分"string"，预处理器就把它作为当前文件的名字。
  • 这条指令将修改__LINE__符号的值，如果加上可选部分，它还将修改__FILE__符号的值

三. #progma

语法因编译器而异，用于支持编译器的特性。 从本质上来说，#progma是不可移植的

四. #

无效指令。 这类指令只是被预处理器简单地删除。
