五. 使用游标

  • 创建游标

      # DB2、MariaDB、MySQL、SQL Server
  DECLARE CustCursor CURSOR
  FOR
  SELECT * FROM Customers
  WHERE cust_email IS NULL;

  # Oracle、PostgreSQL
  DECLARE CURSOR CustCursor
  IS
  SELECT * FROM Customers
  WHERE cust_email IS NULL;
  • 使用游标
    • 打开游标
      OPEN CURSOR CustCursor
    • 访问游标数据 使用FETCH语句
    • 关闭游标
      CLOSE CustCursor
