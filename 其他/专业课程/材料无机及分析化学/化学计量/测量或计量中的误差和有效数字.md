# 一. 误差分类

## 1. 系统误差

**系统误差：由某种固定因素引起的误差**

> - 特点：重复出现、正负及大小可测，具有单向性
>
> - 分类：
>   - 方法误差：由所选择的方法本身决定。
>   - 仪器误差：由仪器本身不够精确而造成 。
>   - 试剂误差：试剂误差来源于试剂不纯。
>   - 操作误差：操作者本人所引起的。
> - 消除方法：根据不同分类选择，如方法误差就更改方法；仪器误差就更换仪器或测出误差值进行运算；试剂误差就更换试剂。



## 2. 偶然误差

**偶然误差：又叫随机误差，由随机的偶然的原因造成**

> - 特点：客观存在，不可避免，大小符合统计规律，双向性
> - 消除方法：多次测量取平均值



## 3. 过失误差

**过失误差：指明显与事实不符的误差，即异常值，亦称“错误”**

> - 消除方法：谨慎操作，多次测量进行判断



# 二. 误差的表征与表示

## 1. 绝对误差与相对误差


- **绝对误差($$E$$)**：$$E=x-\mu$$
	
	> - 测量值($$x$$)：某一次的测量数据。
	> - 真值($$\mu$$)：某一物理量本身具有的客观存在的真实数值。
	
- **相对误差($$E_r$$)**：$$E_r={E\over \mu}\times 100\% = {x-\mu \over \mu}\times 100\%$$

- **平均值($$\overline{x}$$)**：$$n$$次测量数据的算术平均值。

- **准确度**：在一定测量精度下分析结果与真值的接近程度。



## 2. 加标回收率

**加标回收率($$P$$)**：$$P={A-B\over D}\times 100\%$$

> - $$A$$和$$B$$分别为加标试样和未加标试样测定值；D为加标量。
> - 其中$$A$$、$$B$$、$$D$$可以是浓度或所含样品的物质的量
> - 通常$$95\%\sim105\%$$作为回收率的目标值
> - **注意**：
>   - 加标物质的形态应该和待测物的形态相同；
>   - 一般情况下样品的加标量应为样品浓度的$$0.5\sim 2$$倍。



# 三. 偏差的概念与表示

- **精密度**：同一条件下重复分析均匀样品所得测定值的一致程度 。

- **精密度表示方法**：

  - 绝对偏差和相对偏差

  - 平均偏差和相对平均偏差
  - 标准偏差和相对标准偏差
## 1. 绝对偏差和相对偏差

- **绝对偏差($$d$$)**：$$d=x-\overline{x}$$

- **相对偏差($$d_r$$)**：$$d_r={x-\overline{x}\over \overline{x}}\times 100\%$$



## 2. 平均偏差和相对平均偏差

- **平均偏差($$\overline{d}$$)**：$$\sum_{i=1}^n{\vert d_i\vert \over n }$$
- **相对平均偏差($$\overline{d_r}$$)**：$$\overline{d_r}={\overline{d}\over \overline{x}}\times 100\%$$



## 3. 标准偏差和相对标准偏差

### 1）样本（$$n<30$$）

- **样本标准偏差($$S$$)**：$$S=\sqrt{\sum_{i=1}^n{d_i^2}\over n-1}$$
- **样本相对标准偏差($$RSD$$)**：$$RSD={S\over \overline{x}}\times 100\%$$

### 2) 总体（$$n>30$$）

- **总体标准偏差($$\sigma$$)**：$$\sigma =\sqrt{\sum_{i=1}^n{d_i^2}\over n}$$
- **变异系数($$CV$$)**：$$CV={\sigma \over \overline{x}}\times 100\%$$



# 四. 准确度与精密度的关系

准确度与精密度可能的关系有以下四种：

1. 精度高且准确度也好
2. 精度不高但其平均值的准确度仍较好
3. 精度很高但明显存在负的系统误差
4. 精度很差，且准确度也很差，不可取

**注意：**精密度高不一定准确度好（可能有系统误差），而欲得高准确度，必须有高精密度。**常量分析要求相对偏差$$\leqslant 0.2\%$$**。



# 五. 提高分析结果准确度的方法

- 选择合适的分析方法
- 减少系统误差的方法
  - 对照试验：检验和消除方法误差
  - 校准仪器：消除仪器误差
  - 空白试验：消除试剂误差
- 减少偶然误差的方法：增加测定次数
- 控制测量的相对误差：称量重量和移液体积越大，相对误差越小。



# 六. 有效数字及运算规则

## 1. 有效数字的意义

**有效数字的意义：实际能测量到的数据，分析化学中指准确测得的数字加一位不确定数字**



## 2. 有效数字位数的确定

**有效数字位数的确定：所有的确定数字再加一位不确定数字，从第一位非“0”的数字开始推算**

> - 第一位非$$0$$数字前的”$$0$$“其定位作用，不是有效数字
>
> - 第一位非0数字后的“$$0$$”表示测量的精度，是有效数字
>
> - 单位改变，有效位数不改变。
>
>   > 例：
>   >
>   > $$22.00mL$$和$$0.02200L$$都是$$4$$位有效数字
>
> - $$pH$$、$$\lg K$$等对数值的有效位数仅仅取决于其小数点后数字位数，整数部分只起定位作用，不作为有效数字。
>
>   > 如：
>   >
>   > $$pH 12.00$$：$$2$$位有效数字
>   >
>   > $$\lg K=4.76$$：$$2$$位有效数字
>   >
>   > $$K=1.7\times 10^{-5}$$：2位有效数字



## 3. 有效数字的运算规则

- 运算过程遵循"先修约、后计算、再修约"的规则

  > - 先修约：指计算前先将各运算数修约为规定的有效位数
  >
  > - 再修约：值计算后将所得结果修约为规定的有效位数

- 数字修约依据“四舍六入五成双”

  > - 四舍：待修约位数的后一位如果$$\leqslant 4$$，则将后续数字舍去
  >
  > - 六入：待修约位数的后一位如果$$\geqslant 6$$，则将修约的末位进$$1$$，然后将后续数字舍去
  >
  > - 五成双：若待修约位数的后一位$$=5$$，则先看$$5$$后面的数字：
  >   - 若含有非零数字(不一定紧跟5)：则将修约的末位进$$1$$，然后将后续数字舍去
  >   - 若不含有非零数字(后续所有位数都不包含)：则看$$5$$前面的一位：
  >     - 若是奇数，则将修约的末位进1，然后将后续数字舍去
  >     - 若是偶数：则将待修约位数的后续数字舍去

- 若某数字的首位数字$$\geqslant 8$$，则改有效数字的位数可多计算一位。

- 误差和偏差，一般只要求$$1\sim 2$$位有效数字；化学平衡的计算有效位数多为两位。

- 加减法：有效位数以绝对误差最大的数为准，即小数点后位数最少的数字为依据。

- 乘除法：有效位数以相对误差最大的数为准，即有效位数最少的数字为依据
